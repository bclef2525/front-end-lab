# Client-Side Rendering (CSR) Example 🌐

이 예제는 Next.js에서 클라이언트 사이드 렌더링(CSR)을 구현하는 방법을 보여줍니다.

## 주요 특징 ✨

1. **'use client' 지시어**

   - 컴포넌트를 클라이언트 컴포넌트로 지정
   - 브라우저에서 렌더링 수행

2. **데이터 페칭**

   - useEffect를 사용한 클라이언트 측 데이터 페칭
   - 로딩 상태 처리

3. **인터랙션**
   - 클라이언트 측 상태 관리
   - 사용자 이벤트 핸들링

## 코드 설명 📝

```typescript
"use client";

// 클라이언트 컴포넌트 선언
export default function CSRPage() {
  // 클라이언트 측 상태 관리
  // 데이터 페칭 로직
  // 이벤트 핸들러
}
```
